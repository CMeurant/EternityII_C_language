[FR]##################################################################################

Projet développé dans le cadre de ma deuxième année de Licence à l'Université de Bourgogne dans le langage C.

Jeu reproduisant le principe du jeu Eternity II, un puzzle dans lequel même les bords doivent trouver une corrélation avec une autre pièce.
Ce jeu se joue directement sur le terminal, après compilation et exécution du main.
Le menu se veut relativement explicite concernant la marche à suivre pour jouer au jeu.

Ce projet comporte une partie facultative; une "IA" permettant de résoudre le puzzle à la place du joueur dans un temps plus ou moins important en fonction de la taille du puzzle. ( Jusqu'à environ 7 secondes )

Chaque fonctionnalité est détaillée dans le code-source pour favoriser sa lecture et la notation du projet. Amusez-vous bien !

( My apologies, I'm not a native english speaker. Feel free to text me if anything is wrong. )

Project developed for my studies in computer sciences ( Bachelor - 2nd year ) in C language.

This game is Eternity II, a puzzle in which even the puzzle's edges have to find a correspondence with another piece.
You can play directly in the terminal, after compiling and executing the main program.
This game's menu is kinda explicit so you don't have to find a manual.

This projects contains an optional feature; an "AI" able to solve the puzzle alone in between 1 & 7 seconds considering its size ( 7*7 at maximum )

Each functionallity has been detailed within the source-code. Enjoy trying this program !
